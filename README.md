# geojson-normalize

Normalize any GeoJSON object into a GeoJSON FeatureCollection.

## install

```sh
npx jsr add @birkskyum/geojson-normalize
```

## import

```ts
import {normalize} from "@birkskyum/geojson-normalize";
```

## use

```ts
const featureCollection = normalize(object);
```
